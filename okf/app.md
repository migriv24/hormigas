The **app manifest** for this bundle — Hormiga's structured self-description, in the
shape Void Core's app-manifest concept defines. A tool presenting Hormiga from the
outside (a launcher, FaultSack, another Void Core app) reads this instead of scraping
prose. The palette is the app's default theme from `static/css/app.css` (`:root`);
the app ships multiple user-selectable themes, described in
[design language](/ui-ux/design-language.md).

Hormiga is Spanish for "ant": small, organized, built for collaborative work. The
name system extends the metaphor — the [Antfarm](/concepts/antfarm.md) (backend
topology), the planned Studio/Mound/Nest/Queen/Courier features (see
[futures](/references/futures-doc.md)).
