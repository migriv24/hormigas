Hormiga  ·  v—  ·  for LON
Hormiga
A bilingual content management and publishing platform built for the Latine Outreach Network (LON). Manages contacts, events, organizations, images, and job postings — and assembles them into bilingual HTML newsletters. Built around The Antfarm: a node-based protocol system that keeps every backend swappable — local or cloud, your choice.
Python + Flask Electron Local-first The Antfarm Bilingual EN/ES

📋 Data Manager
  • Contacts — people from the Google Sheet. Click any row to edit name, org, email, and tags. Tags are local-only (not written back to Sheet).
  • Events — upcoming events from the Sheet. Each event can have tags and a custom icon set locally.
  • Presenters — presenter sign-up data from the Sheet.
  • Organizations — detected or manually created. Use ⚡ Merge… to combine duplicate entries — select two rows and confirm which name to keep.
  • ↺ Sync (sidebar) — re-fetches all Sheets data and clears the local cache. Run this after editing the spreadsheet directly.
  • Undo / Redo — undoes edits made through the app. Does not undo direct edits in Google Sheets.
📰 Newsletter Builder
  • Sections palette (left panel) — drag onto the canvas to add, or double-click to append. Use ☰ to collapse.
  • Canvas (center) — your newsletter layout. Drag the ⠿ handle to reorder cards (drop on top half = above, bottom half = below).
  • Section editor (right panel) — edit the selected section. Use ⚙ to collapse. For Narrative sections, the Customize button opens a full rich text editor.
  • Preview — renders the full newsletter. Export downloads EN + ES HTML files.
  • Language — set to Español before export to auto-translate all text fields.
  • Projects — save and reopen canvas layouts from data/projects/.

🖼 Images & Fliers
  • Upload — images are hosted on ImgBB. The URL is saved locally in data/images.json.
  • Tags — add tags to any image for filtering. Use the tag filter in the builder to load sets of images.
  • Language pairing — mark an image as EN or ES and then use Pair to link the two versions. When the builder loads flyers by tag, it picks the version that matches the newsletter language automatically.
  • Event linking — link an image (e.g. a flier) to an event using the calendar badge on the image card. Linked images appear in the event's detail view.
  • 📅 badge — shown on image cards that are linked to at least one event. Click the badge to view or edit the linked event.
🔗 Connections Manager
  • ↻ Sync Sheet — imports People, Orgs, Events, and Images as graph nodes. Run after adding new data to any tab.
  • Entity list — filter by type (People, Orgs, Events, Images, Tags). Click any row to see details and connections.
  • Tags filter — shows graph tag nodes. Assign tags to entities from the detail panel.
  • ⚙ Tag Manager — shows ALL tags across every system (images, events, contacts). Click a tag to preview which entities use it. Rename or delete tags with cascade.
  • Add connection — link any two entities with a named relation. The target field autocompletes from all known graph entities.
  • stale badge — entity was removed from the Sheet but its graph connections are preserved.

🔭 On the Horizon
These features are ideas under active consideration — not yet in development. Current focus is The Antfarm (local-first protocol node system).
🧱 Hormiga Studio
Block-based newsletter builder — assemble content by stacking blocks like Scratch, replacing the current section form editor.
🏔 Mound
Static site builder — generate a public-facing website from your org's data and deploy it via an Antfarm output node.
📓 The Nest
Obsidian-style connected notes wired into the tag and entity systems — notes can reference contacts, events, and orgs directly.
👑 The Queen
LLM protocol node — connect a cloud model (Claude, GPT) or a local model (Ollama) to analyze notes and draft newsletter content.
🗺 Territory
Geospatial layer — associate contacts, events, and incidents with map coordinates. Enables location-aware personalized newsletters.
📬 The Courier
Newsletter dispatch — send personalized emails to your contact list directly from Hormiga via a transactional email Antfarm node.
